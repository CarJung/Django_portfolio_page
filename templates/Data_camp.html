Warsaw-housing-market-analysis

About Project

This is a case study made for Data Scientist Associate certification. The goal of this project was to analyze the dataset of gym clients and point out factors that contribute to higher participation on classes. Beside that i have made a model that would predict the participation on classes. The model was made using Scikit learn. The results were visualized using Matplotlib.
